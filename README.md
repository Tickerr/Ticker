# Ticker

Live prices & P&L for any Robinhood Chain wallet — no login, no keys, straight from the chain.

This is a fully client-side static app: `index.html` is the whole product.

**Hosted with GitHub Pages.** To run your own copy: fork or download, then enable
Pages (Settings → Pages → Deploy from a branch → main, root).

Not affiliated with Robinhood. Not investment advice.
